To use:

A) If you only want to run the A* algorithm for the puzzle without visualization, you can do it with the following command:

	clisp AEstrella.lisp n1 ... n16 n1 ... n16

	Alternatively:

	Chmod 755 AEstrella.lisp && ./AEstrella.lisp n1 ... n16 n1 ... n16

	Where the first 16 n's describe the initial state of the puzzle and the following 16 		represent the final or goal state:

	n1  | n2  | n3  | n4
	--------------------
	n5  | n6  | n7  | n8
	--------------------
	n9  | n10 | n11 | n12
	--------------------
	n13 | n14 | n15 | n16
	
	Note: to use those commands, the prompt has to be in the directory: puzzle -> data

B) If you want to use the algorithm with the interactive visualization:

	1. Open the app: "puzzle" in the "aplicaciones" and then the folder corresponding to your OS.
